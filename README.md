# リアルタイム制御向けC++制御モデルライブラリ

## 特徴

- マイコン実装向けC++行列ライブラリを活用して、制御アルゴリズムを効率的に実装可能
- Python Controlの制御アルゴリズム計算コードから簡単に置き換えられる
  - 生成AIを用いて変換を自動化できるように

### 対応言語バージョン

C++11

## 実装済み機能

- 離散時間の伝達関数モデル
- 離散時間のPID制御器
- 離散時間の状態空間モデル
- 線形二次レギュレータ (Linear Quadratic Regulator, LQR)
- カルマンフィルタ
- 最小二乗法
